What I've Built

01 Featured

Host-Based Intrusion Detection System (HIDS)

A full-stack intrusion detection system built from scratch. The Rust agent monitors file integrity, detects brute-force attempts, and flags suspicious processes in real-time using Linux inotify — achieving sub-1-second detection latency. A Sigma-based detection engine maps threats to MITRE ATT&CK tactics, and a WebSocket dashboard provides live alert triage.

Real-time inotify monitoring: latency reduced from 30s → <1s
17 Sigma rules mapped to 8 MITRE ATT&CK tactics
80% detection rate across 10 simulated attack scenarios
Multi-host monitoring via centralized server
